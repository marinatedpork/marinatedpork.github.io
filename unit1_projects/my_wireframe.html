Wireframed

July 24, 2014

The first thing I did when redesigning my index page was ask my "how do I want my user to feel on my page?". My answer: curious, impacted. I wanted something with power. I wanted something with a "Landing Page" feeling. I wanted the use of color to be minimal yet very pronounced. I wanted to use my own photos. So, that is what I created

I use a lot of oversized post-it pads, so the first part of my wireframe was without a doubt hand-drawn. I benefit a lot from the fluidity of the hand-drawn brainstorming process. See below for my goofy phase 1 wire frame:

Pretty simple layout plans in that photo. Next I wanted to sketch out how I was going to display links to my work on my website, so I sketched up a quick CSS wire-frame for the pigs, descriptions and how I was going to implement them:

As you can see I applied my new found display knowledge buy wrapping a couple in-blocks in a block to make for smooth layout planning.
